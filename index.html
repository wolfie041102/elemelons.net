The Elemelons server:

Welcome to the Elemelons server!

The Elemelons server is a survival Roleplay server for Minecraft, created by Wolfie041102 and TheAdoneus. On this server you will find a community that is absolutely addicted to Minecraft, and quite commited to the roleplay. The roleplay you will see on this server is NOT scripted, except for some general outlines on the main story. Feel free to join, build a base, meet the characters, and get involved in the roleplay! And most importanly, Have Fun!

resourcepacks

This server uses a modified version of the bettervanilla recourcepack. We have kept in only the armor, for roleplay and lag purposes. When naming the armor something different, it will show up different. To know what armors there are, visit their server; play.bettervanillabuilding.com. This texturepack is not essential, but it would elevate your roleplay experience. The download link for the pack can be found here

During roleplay we may use some custom resourcepacks to add custom sounds/music to the game. When thats the case and you want to particpate in the roleplay scene make sure to download them for a better experience. The download links for the custom packs can also be found here.
